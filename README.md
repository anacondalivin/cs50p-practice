# cs50p-practice 🐍

My CS50P Python practice code — a collection of beginner Python programs
built while learning the fundamentals.

## Projects

| File | Description |
| `odd-even.py` | Checks if a number is odd or even using boolean functions |
| `simple-thermostat-converter` | Converts Celsius to Fahrenheit |
| `team-gen.py` | Basketball player team lookup |
| `test-results.py` | Grade calculator using if/elif/else with custom messages and f-strings |
| `tip.py` | Tip calculator |
| `variables.py` | Variables practice |
| `playback.py` | Playback program |

## How to Run Any File
```
python filename.py
```
